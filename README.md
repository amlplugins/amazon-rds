# @amlplugins/amazon-rds

Native `@aws-sdk/client-rds` SDK re-exported under the `@amlplugins` namespace.

> Amazon RDS — managed PostgreSQL/MySQL/MariaDB/Oracle/SQL Server/Aurora DB instances, clusters, snapshots.

## Install

```bash
npm install @amlplugins/amazon-rds
```

## Use

```ts
import { /* named exports from @aws-sdk/client-rds */ } from "@amlplugins/amazon-rds";
// or
import sdk from "@amlplugins/amazon-rds";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
